## Creator Toolkit

一个 Claude Code 技能包·覆盖完整的内容生产管线：

```
找选题 → 多平台文案 → 口播脚本
```

### 三个 Skill

| `/topic-finder` | `/content-adapter` | `/script-writer` |
|---|---|---|
| 实时搜索热点·输出选题建议 | 一个选题·三版平台文案 | 按时长生成口播脚本 |
| 3-5个选题卡片 | 抖音 / 小红书 / B站 | 15秒 ~ 5分钟 |

### 快速开始

1. 把 `skills/` 复制到你项目的 `.claude/skills/`
2. 编辑 `config/source-config.md`·改成你的领域
3. 在 Claude Code 里说：

```
/topic-finder 美甲 找3个选题
/content-adapter 夏日果冻美甲合集
/script-writer 果冻甲测评 30秒
```
