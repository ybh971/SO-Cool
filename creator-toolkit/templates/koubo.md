# 口播脚本格式规范

## 核心原则

画面、台词、时长三位一体。每一秒都要有事可看、有话可说。

---

## 时长与结构

### 15秒（极短）

```
0-3秒  钩子：最炸的画面 + 一句话抓人
3-12秒 主体：快速展示核心信息
12-15秒 收尾：引导互动
```

总台词: 40-60字 | 语速: 偏快

### 30秒（标准短）

```
0-3秒   钩子：画面冲击 + 一句话悬念
3-10秒  展开：2-3个信息点快速过
10-25秒 高潮：对比/反转/结果展示
25-30秒 收尾：引导互动 + 关注理由
```

总台词: 80-120字 | 语速: 中快

### 60秒（标准中）

```
0-5秒   钩子：建立期待
5-15秒  铺垫：背景/问题
15-45秒 主体：核心内容（可分2-3个小段）
45-55秒 高潮/结论
55-60秒 收尾：互动引导
```

总台词: 180-250字 | 语速: 中

### 3分钟（长内容）

```
0-10秒    钩子
10-30秒   引入：为什么要聊这个
30-120秒  主体（分章节）:
           - 段落1: xxx (30s)
           - 段落2: xxx (30s)
           - 段落3: xxx (30s)
120-160秒 总结/升华
160-180秒 结尾CTA
```

总台词: 500-700字 | 语速: 中 | 适合B站/视频号

### 5分钟（深度）

```
0-15秒    钩子
15-45秒   引入+背景
45-240秒  主体（4-5个章节）:
           - 每章节30-50秒
           - 章节间有转场
240-280秒 总结
280-300秒 结尾CTA
```

总台词: 800-1100字 | 语速: 中慢 | 适合B站深度内容

## 脚本输出格式

统一使用四列表格：

| 时间 | 画面 | 台词 | 备注 |
|------|------|------|------|
| 0-3秒 | 成品特写 | "只要9.9的果冻甲..." | 语速快，惊讶语气 |
| 3-8秒 | 三款平铺对比 | "我买了6款回来测" | 手势指向 |
| ... | ... | ... | ... |

---

## 画面设计原则

- **0-3秒必须视觉冲击**：成品效果、对比画面、夸张表情
- **信息点配特写**：说细节的时候画面必须跟上特写
- **转场节奏匹配语速**：快语速 = 卡点切，慢语速 = 平滑过渡
- **文字叠加**：关键数字/结论可打文字贴纸覆盖在画面上
- **避免静态画面超过3秒**：要么画面在动 要么切镜头

---

## 台词写作原则

- **口语化**：像跟一个人说话，不是念稿
- **短句**：一口气能说完的长度，不超过15字
- **每段台词对应一个画面变化**：说了新信息 → 切新画面
- **互动设计**：
  - 15/30秒: 结尾一句话引导（"左下角告诉我"）
  - 60秒: 可以中间埋一个互动点 + 结尾一个
  - 3/5分钟: 每章一个互动钩子（"看到这里的扣1"）

---

## 不同内容类型的节奏

| 类型 | 节奏 | 画面策略 |
|------|------|---------|
| 测评/对比 | 快→中→快 | 多角度展示 → 细节特写 → 对比画面 |
| 教程 | 中→慢→中 | 全景展示 → 步骤特写 → 成品效果 |
| 开箱/vlog | 慢→中 | 悬念铺垫 → 逐步揭示 |
| 观点/吐槽 | 快 | 表情+文字卡点 |
| 故事/叙事 | 慢→中→高潮快 | 铺垫→推进→反转 |

## 发布平台适配

| | 抖音 | B站 | 小红书 |
|---|---|---|---|
| 最佳时长 | 15-30秒 | 3-5分钟 | 30-60秒 |
| 画面节奏 | 快 | 中慢 | 中 |
| 口播风格 | 情绪驱动 | 信息驱动 | 精致感+干货 |
